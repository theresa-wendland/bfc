First 7 Command lines to learn

Mac Os

Theresa Campbell
April 14,2019

Follow

When I first opened the terminal It looked scary, and I was worried that I would break my computer which I later found out was possible. From the terminal, you can run a command that deletes all your files including your operating system, not something you would want to mistakenly do, don’t be too alarmed the chances of you doing this is pretty low especially after you finish reading this post.
If you plan on becoming a web developer the terminal is a useful tool to understand. There are several reasons to learn how to use the terminal, the most important being greater control over system functions, such as browsing to a file that is hidden from the graphical user interface (GUI) or scaffolding out projects using command lines like create-react-app. Once you are familiar with the basics you can learn more complex commands.

Let’s dive into the basics

- What is the terminal? The terminal is the interface that allows you to communicate with your computer. You write a command in the terminal and your computer will carry out your command. There are many commands that the terminal understands. There is even a command that allows you to play old-school arcade games. The command lines that we will be going over are for MacOS, tutorials for PC, Linux are coming soon.

Where to find it

The Terminal app is in the Utilities folder in Applications. To open it, either open your Applications folder, then open Utilities and double-click on Terminal; or, the way I usually find it is by pressing Command - spacebar to launch Spotlight and then type "Terminal," and then double-click the search result “Terminal”.

7 Basic Command lines

pwd - Print Working Directory

This command shows what directory/folder you are currently in. We are currently on our desktop. This is important because it lets you know what directory you are working in.

cd - change directory

This command takes you to a specific directory/folder. This is important because it allows you to move from one directory to the next. You can use cd ../ to go back one folder, this is similar to a back button. I have changed directories and I am now in my gallery directory/folder.

ls- List

This command lists all the files in your current directory. This will help you see all of the files in a particular directory. Let’s take a look at all the files in my gallery folder.

mkdir - make a directory

This command allows you to make a directory/folder. If you’re anything like me, the first time you do this, you will be really impressed with your new powers, because you can now create a new folder without using the GUI. You are already coding! Let’s make a new directory/folder called, “Art”. If you run the ls command after you have made your new directory you will see the new directory/folder that you just created.

rmdir - remove directory

This command allows you to delete a directory/folder. Be careful with this command if you are new to using the terminal. You could accidentally delete a file. It won’t ask you if “you are sure you want to delete this file?”, just boom and its gone.

touch

This command allows you to create a new empty file. This is a good shortcut to create a new file which is the starting point for most projects. Change directories, navigate to the Art folder that you just created and created a file called index.html touch index.html

nano

This command allows you to create a new file and immediately have the ability to edit the contents of this file. The nano command launches a text editor. This is an even better shortcut than “touch” if you are actually working in the nano text editor. Otherwise, use “touch”. Let’s make a heading for our web page. When you are finished press command X to exit the terminal and yes to save your changes. nano home.html Nice work! You are now familiar with the terminal. You can find your location, navigate to a new directory, create a new directory, add, delete and edit files.

Quick tips

Pressing the up row will show you your most recent commands.
Clear, will remove previous commands from the terminal.
